In our first exercise, we will design a so called one pager site. One pagers are websites from a single page. Most of them has a single purpose: to make people to do something. It is Call to action.

Subtitle

Select a project from kickstarter.com The purpose of the website is to make people to back the project.

Here are some guidelines:

  1. Select a real project, not a dream. This is a dream: Click Here This is a project: Click Here Too
    What is the difference?
  2. The ukulele is ready, there is a target audience and a price for the product. Select a project with considerable content. Example: https://www.kickstarter.com/projects/clickpackpro/snappack-travel-commute-and-anti-theft
    There is a lot of explanation about the project and high resolution images from the product without any graphics and text on them. Tip: check the official website of the product for high resolution images
  3. Select a right kind of project There are two kind of products, like iPhone (https://www.apple.com/iphone/) and exquisite tea (https://exquisiteleaves.com/). The iPhone is designed to sell itself, it look luxury and elegant and expensive. The tea itself looks like some suspicious weed. So, if you select the first kind, the images of the product maybe enough to make the webdesign. If you want to sell tea, the images of the tea don't work. Yo have to get images of a freshly made tea and happy people consuming it. Or make drawings, whatever.
